My Design Process

My design process starts with understanding. It is about searching the right question and the problem that needs to be solved. We are designing for businesses and people. We have to think about both of them while focusing on the problems.

Empathize

Empathy help we understanding about the user to define the problem is.

Define

From this, we can define the problem in the clear way.

Ideate

List down ideas to solve these problems.

Prototype

From the ideas, bring them to Prototype, visualize them to the real interaction.

Test

Do the testing base on those Prototype, from this can learn about the users through testing, reveal insights that can re-define the problem then create new ideas.
